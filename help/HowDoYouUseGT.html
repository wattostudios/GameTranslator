How Do You Use Game Translator

Back to Index

Open a language pack from a game?

Click the "Open Language Pack" button at the top of the screen - the SidePanel changes to show a list of files and folders.
Locate the file you want to open, and double-click it.
Game Translator will look at the language pack, and determine if a plugin can open it. If so, the language pack will be opened, and the TextList will be filled with information.


Open Language Pack


Change the texts (do the translation)?

The TextList has at least 2 main columns - the Original and the Translated text. When you open a language pack, these columns will both have the same values.
Click the first text in the Translated column, and it will allow you to change the value. Pressing Enter or Tab will move to the next text that you need to translate. The table columns that can be edited have a small icon in the column heading
When you change a text, the icon in the Renamed column will change, so you know which texts have been translated already.

Save a language pack, and use it in the game?

Click the "Save Language Pack" button at the top of the screen, or at the bottom of the SidePanel.
If the language pack you opened can be saved, the controls will change to show the saving options.
Supply the name of your new language pack, and click the Save button.
