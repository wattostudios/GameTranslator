The Game Translator Interface

Back to Index

The interface of Game Translator only contains 3 main areas, which keeps it simple, and removes the need for multiple screens and large popup boxes.

1. The big box in the middle is the TextList. When you open a language pack, the TextList will display the original, translation, and other details of every text in the language pack. Whenever you need to deal with specific texts, or the language pack as a whole, you will do it here.

TextList


2. The most important and powerful section is the box on the right: the SidePanel. The SidePanel is used to perform all the main functions of Game Translator, including opening and saving language packs. No matter what you need to do, the SidePanel will always be the place to go.

SidePanel


3. The buttons and menus at the top load different screens into the SidePanel. For example, if you want to open a language pack, click the 'Open Language Pack' button and the SidePanel will change. You can also perform the same thing using the menus.

Buttons and Menus


Depending on the type of language pack, there may be additional properties that can be set on an archive, such as version numbers and descriptions. If a language pack has additional properties, they will be displayed at the bottom of the TextList.
