Searches for texts in the language pack.

In the input field at the top, write the value that you are searching for. Under that, tick the fields that you want to search in.


Search Button

There are 2 types of searches that you can perform...

Find Next Match: Finds the next matching text in the language pack, and selects it.

Select All Matches: Selects all the texts that match the search text.
