Options

Back to Index

Allows you to set a variety of Game Translator properties. Choose a option group from the box at the top, and the SidePanel will change to show all the options in that group. When you have finished changing the options, click the Save Options button at the bottom.


Options Button

The options are categorized as follows:

Appearance: Change the color and appearance of the program
Directories: The directories used for various things such as temporary files and languages.
General: Basic options that don't fit in to any other category
Opening Language Packs: Settings for opening language packs, and for affecting the way Game Translator determines the correct plugin for opening a language pack.
Popups: Enable or disable the popups that appear when performing a task
Table Columns: Choose the columns that are shown in the TextList on the left
Toolbar Changer: Change and re-order the buttons that are shown on the toolbar

If you move the mouse over an option, a more details description of the option will be shown in the statusbar at the bottom of the screen.
